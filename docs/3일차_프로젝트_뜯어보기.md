### app/main.py

FastAPI 애플리케이션의 시작점이다.

현재 프로젝트에서는 `app = FastAPI()`로 앱을 만들고, `practice_router`를 `app.include_router()`로 등록한다. 또한 `/static`, `/media` 경로를 mount하여 정적 파일과 업로드 파일을 제공한다.

서버 실행 시 사용하는 경로도 이 파일의 `app`이다.

```bash
uv run uvicorn app.main:app --reload
```

### app/core/config.py

환경변수를 읽어 프로젝트 설정값으로 사용하는 파일이다.

현재 프로젝트는 `pydantic-settings`의 `BaseSettings`를 사용한다. `.env` 파일에 값이 있으면 해당 값을 읽고, 없으면 클래스에 정의된 기본값을 사용한다.

현재 관리하는 주요 값은 다음과 같다.
   
```
DB_USER
DB_PASSWORD
DB_HOST
DB_PORT
DB_NAME
```
